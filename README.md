# Android MistSDK      

## Overview
Mist SDK will provide you the indoor bluedot experience using Mistâ€™s 16 vBLE antenna array Access point. Using this SDK you will know where the user is and can provide proximity related notification using Mist patented vBeacon technology.

## Beta Release:    

Mist vBLE Android SDK is also distributed as .aar file. [[Download Mist SDK]](https://github.com/mistsys/mist-vble-android-sdk/tree/beta/Library) 

Add following into App/build.gradle

```
implementation(name:'mistsdk-framework', ext:'aar')
implementation(name:'mist-mobile', ext:'aar')
```
